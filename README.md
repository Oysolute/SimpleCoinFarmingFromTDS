# SimpleCoinFarmingFromTDS

A very simple AHK to farm money in the game Tower Defense Simulator, you can choose modifiers, auto-start, or just disable the modifiers

# What modifiers?

This AHK Supports all 13 modifiers from the place : Hidden Enemies, Glass, Exploding Enemies, Limitation Makes Creativity, Committed, Healthy Enemies, Speedy Enemies, Quarantine, Fog, Flying Enemies, Broke, Jailed Towers and Inflation

# How this Work

This script contains all the button coordinates, which fully optimizes entering and setting up the game itself. With the correct settings, the script will enter the place on its own, select modifiers by itself, skip waves, and restart the game, BUT it can fix a place crash, which might worsen the game. To activate the menu, you need to press "F1", to start it press "Home", and to close it press "End". There is also a possibility to check your cursor coordinates by pressing F2, which may be useful for those who also want to improve the script without using third-party applications to determine button positions and press "CTRL+SHIFT+C" you can put the window in the center of the screen

# What me need

You need to set the Roblox size to approximately 802x632 and place it in the center of the screen. After the last update, you just need to select the Roblox window and press CTRL+SHIFT+C, and the window should center (you can also do this with other applications). Then you just need to select the settings for yourself and leave the device alone.
